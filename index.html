Versionamento

O git é um software que permite realiuzar versões de um projeto. O github descktop é um software que faz a ligação entre visual studio (vscode), git e github.

github desktop permite fazer a sincornia entre o repositorio local e o repositorio remoto

commit

É um processo de confirmação de uma versão, ou seja, a cada alteração é possivel confirmar a alteração.

Push

Esse comando envia (sincroniza) as alterações realizadas no repositorio local com o repositorio remoto no github.
